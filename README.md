# 🧬 Breast Cancer Detection using Machine Learning

## 📌 Overview

This project is a web-based application that detects breast cancer from input images using a trained deep learning model. It helps in early diagnosis by classifying images as cancerous or non-cancerous.

## 🚀 Features

* Image-based cancer detection
* Flask web application for user interaction
* Pre-trained deep learning model
* Simple UI using HTML/CSS

## 🛠 Tech Stack

* Python
* TensorFlow
* Flask
* OpenCV
* NumPy

## 📂 Project Structure

* app.py → Flask web app
* retrain.py → Model training script
* label_image.py → Prediction script
* templates/ → Frontend HTML files
* static/ → CSS and assets

## 📊 Model

* Pre-trained CNN model (.pb file)
* Fine-tuned for breast cancer classification
